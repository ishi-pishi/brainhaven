# brainhaven
a full-stack focus tracker app - pomodoro timer with immersive/customizable themes (bgm, ambience, and background images) that blocks websites for you + a focus tracker to see what kind of work you get done.

To run:
- `cd client`
- `firebase dataconnect:sdk:generate`
- `npm run dev`
- `firebase emulators:start --only auth`
- `firebase emulators:start --only dataconnect`
